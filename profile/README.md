# Nedex

Nedex builds Arabic-first SaaS products for the MENA region.

## Products

- **[Nedexedu](https://github.com/Nedex-Sy/Nedexedu)** — school management platform (web + mobile, RTL-first).
- **[nedex-website](https://github.com/Nedex-Sy/nedex-website)** — the company site at [nedex.net](https://nedex.net).
- _More products to follow under the same brand._

## How we work

- **Three founders, equal say.** Decisions touching security, money flows, or data integrity require 2-of-3 sign-off.
- **Documents are the law. The code obeys.** Permissions, tiers, and roles are written down before features are built.
- **Project management lives in [Notion](https://www.notion.so/) (Project HQ).** Commits to `main` reference Notion tasks via `Closes MS-NN`, which auto-closes them.
- **Append-only money flows.** No `UPDATE` on balances — every transaction is an immutable row.

## Stack

Next.js 15 (App Router) · React Native + Expo · Prisma + Supabase Postgres · JWT (`jose`) · shadcn/ui + Tailwind (Arabic RTL) · Vercel (Fluid Compute) · pnpm.
